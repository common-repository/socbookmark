== Description ==
Socbookmark plugin integrates a table at bottom of single post to make post easy for sharing on bookmark sites




== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Go to the Plugins page and activate the plugin.
3. And then go to single.php in your theme, find <div id="comments"> and right before it paste this code  <?php display_socbookmark(); ?> to display the table. 
4.That's all folks
